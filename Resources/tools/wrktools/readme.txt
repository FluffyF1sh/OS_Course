This folder contains a collection of source code (if applicable) of tools, 
projects, and documentation related to the Windows Research Kernel (WRK) we 
consider useful. The WRK comprises the Windows Server 2003 kernel source code 
and is available only to academic institutions.

This folder contains the following directories:

- systablegen: This project contains a generator that 
	helps creating new system service calls for the WRK.
